# Apollo Prospecting Agent

Find target contacts at named companies and grow your CRM in one pass. Hand the agent a list of company websites and the job titles you want to reach, and it discovers matching people through Apollo, files them as contacts against the right company account, and optionally adds them to a list ready for outreach. Each company account is matched before it is created so accounts stay unique; contacts need a stable identity (email, LinkedIn URL, or Apollo id) and are saved as new records, so re-running over the same companies can create duplicate contacts to clean up later.

## Works with

- Apollo

## Capabilities

- Discover decision-makers at a list of target companies by job title
- Create or match a CRM account for each company in one pass
- Save discovered people as contacts attached to the right account
- Append the new contacts to a list you nominate for follow-up campaigns
- Return a structured summary of hits, saves, list adds, and per-company failures
